# Qadamchi 2.2 - O'rnatish

## Talablar
- PHP 7.4 yoki yuqori
- Composer (ixtiyoriy, lekin tavsiya etiladi)

## O'rnatish
### Avtomatik (Standalone)
```bash
php install.php
```

### Composer bilan
```bash
composer create-project urinboy/qadamchi myproject
```

## Server Ishga Tushirish
```bash
php qadamchi serve
```
Brauzerda: http://localhost:8080

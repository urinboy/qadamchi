# Qadamchi

Qadamchi — yengil, oson va zamonaviy PHP micro-framework.  
O‘rnatish:  
```sh
php install.php
```

Serverni ishga tushirish:
```sh
php qadamchi serve
```

Yangi controller yaratish:
```sh
php qadamchi make:controller TestController
```
